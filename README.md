# Smart Cargo

**A inteligência por trás da rota.**

O Smart Cargo é um aplicativo para preparação inteligente de cargas de entrega.

## Missão

Organizar pacotes por paradas, reduzir erros e aumentar a confiança do entregador antes da rota.

## Princípios

- Precisão acima da velocidade
- Menos é mais
- O app nunca inventa informações
- Uma parada é um local físico
- Um pacote pertence a uma parada

## Objetivo da V1

Permitir que o entregador escaneie etiquetas, agrupe pacotes por parada, gere CSV para o Circuit e salve localizações corrigidas.
